#Levels

### Create a Level

`POST /levels`

* **Required Params:**
  * none

* Response: 
  * Status Code: 201 created

### Get a Level

`GET /levels/:id`

* **Required Params:**
  * none

* Response: 
  * Status Code: 200 ok

### Get All Levels

`GET /levels`

* **Required Params:**

* Response:
